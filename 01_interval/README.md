## ðŸ“– Glossary

### HTML (Structure)
| Term | Description |
| :--- | :--- |
| **Semantic Tags** | Tags like <header>, <main>, and <footer> that explain their purpose to both the browser and the developer. |
| **Elements** | The building blocks of HTML, consisting of a start tag, content, and an end tag (e.g., <p>Text</p>). |
| **Attributes** | Provide additional information about elements (e.g., href in an <a> tag). |

### CSS (Style)
| Term | Description |
| :--- | :--- |
| **Selector** | Used to "target" the HTML element you want to style (e.g., h1 { color: red; }). |
| **Box Model** | The concept that every element is a box consisting of Content, Padding, Border, and Margin. |

### JavaScript (Logic)
| Term | Description | Example |
| :--- | :--- | :--- |
| **Variables** | Containers for storing data values. Use const for values that don't change and let for those that do. | let age = 25; |
| **Data Types** | The different kinds of data (String, Number, Boolean, Array, Object). | "Hello" (String) |
| **console.log()** | A function used to print messages to the developer console for debugging. | console.log(x); |
| **Functions** | Blocks of code designed to perform a particular task. | function greet() { ... } |

## ðŸ›  Best Practices for Beginners
1. **Always use semantic HTML**: Don't just use <div> for everything; use tags that describe content.
2. **Comment your code**: Explain *why* you are doing something for your future self.
3. **Keep the Console open**: JavaScript errors only show up there!
4. **Indentation matters**: Keep your code clean and readable to avoid syntax errors.

## ðŸ–¥ How to Check the Browser Console

To see your JavaScript output (console.log), you need to open the Developer Tools.

### Google Chrome
1. **Right-click** anywhere on the page and select **Inspect**.
2. Click on the **Console** tab at the top of the panel.
3. **Shortcut**: Cmd + Option + J (Mac) or Ctrl + Shift + J (Windows).

### Safari
1. **Enable the Develop menu**: Go to **Safari** > **Settings** (or **Preferences**) > **Advanced** and check **"Show features for web developers"** (or "Show Develop menu in menu bar").
2. **Right-click** on the page and select **Inspect Element**.
3. Click on the **Console** tab.
4. **Shortcut**: Cmd + Option + C (Mac).

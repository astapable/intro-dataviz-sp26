# Intro to Data Visualization - Labs Overview

## 01_interval
Basic D3.js clock visualizations. Learn SVG setup, time markers, and D3 selections.

## 02_d3_bins
Explore binning and histograms with D3.js. Visualize distributions and learn about data grouping.

## 03_d3_bins_scaleOrdinal
Use D3's scaleOrdinal for categorical binning and color mapping in histograms.

## 04_d3_air-quality
NYC Air Quality Visualization. Interactive time-series line chart of PM2.5 levels using D3.js. Features tooltips, responsive SVG, and real-world data from NYC Open Data.

## 05_tmdb-fetch
Fetch and display movie data from TMDB API. Learn about data fetching, parsing, and basic visualization.

## 06_tmdb-gallery
Create a gallery of movie images using D3.js. Sort and display images with metadata from a JSON file.

## 07_netflix-added
Display images with D3.js. Fetch, sort, and visualize Netflix data with images, dates, and titles.

## 08_sentiment
Analyze and visualize text sentiment. Use D3.js to show sentiment scores and text frequency.

## 09_text-frequency
Visualize word frequency in text data. Learn about parsing, counting, and displaying results with D3.js.

## 10_leaflet
Interactive geographic visualization using Leaflet.js and D3.js. Map data points and explore spatial data.

## 11_imdb
Netflix Titles World Map. Map Netflix content by production country using TMDB API and GeoJSON. Interactive filtering and country-level data display.

## 12_imdb_mapping
Advanced mapping of Netflix titles. Visualize production locations and metadata with interactive maps.
